# Chillll

Ordinary music database following [ChillBox](https://github.com/jonathanjoster/ChillBox).

## Setup
```
bundle install
```

## How to run
```
rails s
```

## Requirements
```
$ ruby -v
ruby 3.0.3p157 (2021-11-24 revision 3fb7d2cadc) [x86_64-darwin20

$ rails -v
Rails 7.0.5
```
